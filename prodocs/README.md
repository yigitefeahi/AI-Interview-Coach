# prodocs — Proje referans klasörü

## Bu klasör ne?

`/prodocs`, **yapay zeka geliştirme ajanları** (Cursor, Copilot vb.) ve yeni katılan geliştiriciler için hazırlanmış **kısa referans dokümanları**dır.

Ana PRD veya kullanıcı dokümanı değildir. Amaç: "Bu projede X nasıl çalışıyor?" sorusuna hızlı cevap vermek.

## İçerik

| Dosya | Ne anlatır |
|-------|------------|
| `architecture.md` | Sistem mimarisi, veri akışı |
| `api-overview.md` | Önemli API endpoint'leri |
| `product-context.md` | Ürün özeti, hedef kullanıcı, MVP kapsamı |
| `folder-map.md` | Repo klasör yapısı |

## Ne zaman kullanılır?

- Yeni özellik eklerken hangi dosyaya dokunacağını bulmak
- AI ajanına proje bağlamı vermek ("önce prodocs'u oku")
- Demo veya teslim öncesi mimariyi hatırlamak

## Üst düzey dokümanlar (kök dizin)

- `PRD.md` — ürün anayasası
- `Plan.md` — teknik plan ve user story'ler
- `tech-stack.md` — teknoloji seçimleri
- `DesignSystem.md` — UI kuralları
- `Progress.md` — geliştirme günlüğü
